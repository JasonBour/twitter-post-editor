# Twitter 帖子编辑器

这是一个完全可编辑的 Twitter/X 风格帖子界面，可以自定义所有内容并导出为图片。

## 功能特点

✨ **完全可编辑**
- 用户名、账号、帖子内容都可以直接点击编辑
- 所有互动数据（评论、转发、点赞、收藏数）都可以修改
- 时间和浏览量也可以自定义

🖼️ **图片功能**
- 点击头像可以上传自定义图片
- 支持导出整个帖子为高清图片

💾 **自动保存**
- 所有修改会自动保存到浏览器本地存储
- 刷新页面后内容不会丢失

## 使用方法

1. 直接在浏览器中打开 `index.html`
2. 点击任何文字内容即可编辑
3. 点击头像可以更换图片
4. 使用右侧控制面板的按钮进行操作

## 编辑提示

- **用户名**：点击"文硕"即可修改
- **账号**：点击"@suosimath"即可修改
- **帖子内容**：点击主要文字区域即可编辑
- **互动数据**：点击数字即可修改
- **头像**：点击头像或使用"更换头像"按钮

## 导出图片

点击"导出为图片"按钮，会自动下载一张高清的帖子截图。

## 重置内容

点击"重置内容"按钮可以恢复到初始状态。
